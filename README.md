# SEFormatConvertor
### A simple convertor for Model/Animation formats

## Support formats

||SEModel|SEAnim|SMD|LTB|DTX|
|----|----|----|----|----|----|
|SEModel|||⇌|←||
|SEAnim|||←|←||
|SMD|⇌|→||||
|PNG|||||←|

#### ⇌ Means both way, ← is from and → is to.

Maybe more formats later
